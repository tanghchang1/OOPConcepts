Diagrama del proyecto:






Objetivos del Proyecto para el Centro Deportivo "Los Nevados":
Objetivo General:

Desarrollar una aplicación web de gestión de instalaciones, socios y alquileres para "Los Nevados".
Objetivos Específicos:

Diseñar una interfaz de fácil navegación.
Crear un módulo para registrar y consultar instalaciones deportivas.
Implementar un módulo para registrar y consultar socios.
Desarrollar un módulo para gestionar alquileres, verificar disponibilidad de instalaciones, evitar cruces de horarios y determinar el valor del alquiler.
Considerar la incorporación de funcionalidades adicionales según las necesidades y sugerencias del cliente.
Objetivos Técnicos:

Emplear un diseño orientado a objetos.
Elaborar un diagrama de clases que refleje la estructura del sistema.
Asegurar la compatibilidad web y la calidad del software mediante validaciones.
Objetivos de Gestión:

Entregar documentación completa en el plazo establecido.
Mantener comunicación con el cliente y adaptar el desarrollo según retroalimentación.
Seguir buenas prácticas para garantizar una entrega oportuna y de calidad.

Requisitos para el Centro Deportivo "Los Nevados":
Requisitos Funcionales:
Gestión de Instalaciones:
Registrar y consultar instalaciones con sus características y especificaciones.
Gestión de Socios:
Registrar y consultar socios con su información básica.
Gestión de Alquileres:
Permitir alquileres por parte de socios, verificar disponibilidad de instalaciones, calcular costos y consultar alquileres de un socio.
Requisitos No Funcionales:
Usabilidad:
Interfaz intuitiva y fácil de usar.
Rendimiento:
Respuestas rápidas a acciones de usuario.
Seguridad:
Protección de datos y acceso solo a personal autorizado.
Compatibilidad:
Funcionalidad en principales navegadores web.
Escalabilidad:
Capacidad de adaptarse a un aumento de usuarios y datos.
Mantenibilidad:
Código estructurado y documentado.
Estos requisitos brindan una visión comprensible y concisa de las principales funcionalidades y características que debe tener la aplicación para el Centro Deportivo "Los Nevados".



Alcance  del Proyecto para "Los Nevados":
Funcionalidades:
Instalaciones: Creación, edición y consulta de instalaciones deportivas categorizadas por tipo.
Socios: Registro y consulta de información básica de socios.
Alquileres: Registro con verificación de disponibilidad, cálculo automático de costo y consulta de alquileres por socio.
Características Técnicas:
Aplicación web responsiva.
Mecanismos de seguridad para la protección de datos.
Base de datos para gestionar la información.
Entregables:
Aplicación web testeada.
Documentación técnica y del usuario.
Código fuente con documentación.
Restricciones:
Desarrollo solo en idioma español.
Enfoque exclusivo en instalaciones, socios y alquileres.
Exclusiones:
Sin sistemas de pago en línea o integraciones con plataformas financieras.
No se integrará con otros sistemas del centro deportivo.
Este alcance resumido ofrece una visión general y concisa de lo que se cubrirá en el proyecto de desarrollo para el Centro Deportivo "Los Nevados".

Riesgos del Proyecto:
Cambios de Requerimientos: Posibles modificaciones en las demandas del cliente tras comenzar el desarrollo.
Tecnología y Herramientas: Problemas o incompatibilidades con las herramientas seleccionadas y falta de experiencia del equipo con ellas.
Riesgos de Personal: Rotación de personal clave o falta de habilidades necesarias en el equipo.
Problemas con Infraestructura: Fallos en servidores, base de datos o conexión a internet.
Problemas de Integración: Dificultades al unir diferentes módulos de la aplicación.
Riesgos de Seguridad: Vulnerabilidades que podrían comprometer la información.
Rendimiento Inferior: La aplicación no cumple con las expectativas de rendimiento.
Riesgos Financieros: Costos no previstos que superen el presupuesto o falta de financiamiento.
Limitaciones de Tiempo: Retrasos en el cronograma debido a imprevistos.
Problemas de Calidad: Defectos no detectados que afectan a los usuarios.
Riesgos Externos: Cambios en regulaciones o factores externos imprevistos.
Desafíos de Comunicación: Falta de comunicación clara entre stakeholders y el equipo de desarrollo.
Es fundamental tener estrategias de mitigación para cada riesgo identificado, garantizando una respuesta efectiva en caso de que se materialicen.
